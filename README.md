## Setup

```bash
make up
```

App disponibile su:
http://localhost:8080

## Comandi principali
```bash
make up        # avvia ambiente dev
make down      # ferma ambiente
make test      # esegue test
make lint      # controlli qualità
make format    # formatta codice
make bash      # entra nel container
```
